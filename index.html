2.- Flex Direction

1
2
3

In this part the main idea is to test different options for flex-directions

  • flex-direction: row Is the default option and places the div at the beginning of the parent container in a row
  • flex-direction: row-reverse with this one the order is 3|2|1 but they are placed at the end of the row
  • flex-direction: column with this option the divs are placed in a single column in ascendent order
  • flex-direction: column-reverse with this option the divs are placed in a single column but sorted in descendent order and at the end of the column
